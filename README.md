# ovs-flow-mgmt
OpenVSwitch OpenFlow Flow Management is a Web Based Flow Management for OpenVSwitch. This project is help network engineer deploy rule to OpenVSwitch via dashboard.

### Feature
1. Layer 2 Flow
1. Layer 3 Flow
1. Layer 4 Flow
1. Dashboard
